# Offer Countdown

Componente custom de VTEX IO que muestra un **contador regresivo** para una oferta por tiempo limitado en la Home: días, horas, minutos y segundos. Cuando la oferta termina, se reemplaza por un mensaje de cierre.

Todo lo visible (mostrar/ocultar, texto principal, fecha final y texto de finalización) se configura desde el **Site Editor**.

La guía de instalación, link y configuración está en el [README principal](../README.md).
